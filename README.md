🎮 Anime Quiz Fullstack

Um quiz interativo para fãs de anime testarem seus conhecimentos em universos icônicos como One Piece, Naruto, Dragon Ball Z e Toriko.

👉 Acesse o projeto:
🔗 https://anime-quiz-fullstack.vercel.app/

📌 Sobre o Projeto

O Anime Quiz Fullstack é uma aplicação web que permite aos usuários responder perguntas de múltipla escolha sobre diferentes animes populares.

O projeto foi desenvolvido com foco em prática fullstack, trabalhando tanto o front-end quanto a manipulação de dados no back-end.

Além disso, o app pode ser instalado no celular, funcionando como um aplicativo (PWA).
